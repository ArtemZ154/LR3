# DBMS Lab Project (Go)

Этот файл содержит инструкции по запуску, сборке и тестированию проекта.

## 1. Запуск приложения

Самый быстрый способ запустить проект без предварительной компиляции:

```bash
go run cmd/main.go
```

## 2. Сборка и запуск исполняемого файла

Чтобы создать исполняемый файл (бинарник) и запустить его:

```bash
# Сборка
go build -o dbms cmd/main.go

# Запуск (macOS/Linux)
./dbms

# Запуск (Windows)
dbms.exe
```

## 3. Тестирование

Запуск всех модульных тестов в проекте:

```bash
go test ./...
```

## 4. Проверка покрытия кода тестами (Code Coverage)

Мы добились высокого покрытия тестами (>98%). Чтобы проверить это самостоятельно:

```bash
# 1. Запуск тестов с сбором статистики покрытия
go test -coverprofile=coverage.out ./...

# 2. Вывод отчета в терминал (проценты по функциям)
go tool cover -func=coverage.out

# 3. (Опционально) Открытие детального отчета в браузере
go tool cover -html=coverage.out
```

## 5. Примеры команд внутри программы

После запуска программы (когда появится `> `), вы можете вводить команды базы данных:

- **Массивы:**
  - `MPUSH arr1 10 20 30`
  - `MGET arr1 0`
  - `PRINT`
- **Стеки:**
  - `SPUSH stack1 hello`
  - `SPOP stack1`
- **Очереди:**
  - `QPUSH q1 world`
  - `QPOP q1`
- **Списки:**
  - `LPUSHFRONT list1 A`
  - `LPUSHBACK list1 B`
- **Выход:**
  - `EXIT`
